Thoughts on Progressive Enhancement

Progressive enhancement is the idea that as websites should be able to be used by everyone who comes to them. This is done by using said progressive enhancement by starting with a solid base layer of html that makes the website work for everyone no matter how fancy they like it to be. Then for your next level of user you add css and that takes the website up another step and allows for more creativity while also not being the cream of the crop of design. Finally you get the JavaScript which allows for your highest end users to have the top of the line interaction with the website and all the things that the internet can currently allow users to do on a website with no restrictions to older search engines capabilities.

It is important for the web as without it many users would be forced to stop using top of the line websites as they wouldn't be able to access all the high end javascript stuff and would therefore be limiting the world wide web to only a select group of people. Over this year I have learned that one thing you don't want to do is limit the number of people who can access the public content you make. That's why progressive enhancement is so important.
